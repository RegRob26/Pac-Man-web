Objetivos

 Implementar una adaptación del videojuego Pac-Man, con los conicimientos adquiridos durante las clases de la materia programación web.

Alcance

 Este documento está dirigido hacia quien esté interesado en revisar el código del proyecto y el proyecto en sí mismo. Además de a quienes deseen aportar al videojuego.

Requirimientos

El proyecto se visualizó mediante el navegador Chrome. Por lo que este es un requisito para la visualización, ya que no se garantiza el funcionamiento con otros navegadores web.

Herramientas utilizadas
